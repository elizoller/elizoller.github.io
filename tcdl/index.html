The Project

"Howdy, Mr. President!"

A Fort Worth Perspective of JFK

"Howdy, Mr. President!"

companion online exhibit to the physical exhibit

focus on JFK's time in Fort Worth

showcases over 80 photographs from the Fort Worth Star-Telegram

in the UT Arlington's Special Collections in Fall 2013. depicting JFK's visit to Fort Worth and the events following his assassination in Dallas, November 1963. create a sense of "walking" through the exhibit online if you couldn't make the physical exhibit. preserve the physical exhibit content for future access. have users access the content on their devices while in the physical exhibit

Where to start?

Responsive Web Design

    a web design approach
  • optimal viewing across multitute of devices
  • adapts/responds to the viewing environment

Why RWD?

  • 57% of American adults are cell internet users
  • 34% of cell internet users mostly use their mobile device to access the internet

Pew Internet Study 2013

What makes it different from a mobile site?

No need to maintain multiple sets of content or pages. using a single url allows content to be shared easier and reduces load time by avoiding redirects. allows you set multiple breakpoints for a multitude of devices

How?

  1. a fluid grid
  2. flexible images and media

One Solution: Front-End Framework

  • a pre-packaged set of CSS, Javascript, and HTML
  • to help make development easier and faster
  • aids with browser compatibility
  • rapid prototyping

Introducing Bootstrap

-open source front-end framework

-set of minified CSS, JS, and fonts

-12-column flexible grid layout

-pre-established components for glyphs, buttons, navigation, forms, and more

Pros of Bootstrap

  • pre-designed UI
    • great for developers!
  • highly customizable with CSS-preprocessor Less
  • compatible back through IE-8
  • built-in accessibility support

Cons of Bootstrap

  • Bootstrap sites are "bootstrap-y"
  • large file sizes- unused code

How to use Bootstrap

CSS

The Grid- complete with 4 breakpoints, nesting, offsetting, and ordering. Helper classes for alignment, justification, blockquotes, lists, tables, forms, buttons, and more!. Prepackaged and minified or Mixins with Less to control the grid, breakpoints, colors, fonts, etc.

UI Components

Glyphicons, dropdowns, button groups, navs, navbars, pagination, labels, badges, jumbotron, thumbnails, alerts, progress bars, panels, and wells

JS

Individual or compiled

Transitions, modals, dropdowns, scrollspy, toggleable tabs, tooltips, popovers, alerts, buttons, collapse, carousel, and affix
